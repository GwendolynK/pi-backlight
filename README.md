# Simple perl script for adjusting backlight

Designed and tested for raspberry pi. <br>
 <br>

## Install:

To install: <br>
`$ sudo ./install.sh` <br>
<br>
To uninstall: <br>
`$ sudo ./install.sh -u` <br>
<br>


## How to use:

`$ pi-brightness [15-255]` <br>
`$ pi-brightness` <br>
<br>
For sleep mode: <br>
`$ pi-brightness -s` <br>
<br>
Help menu: <br>
`$ pi-brightness -help` <br>
<br>
