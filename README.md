# Simple shell script for adjusting backlight

Designed and tested for raspberry pi. <br>
 <br>

## Install:

To install: <br>
`$ sudo ./install.sh` <br>
<br>
To uninstall: <br>
`$ sudo ./install.sh -u` <br>
<br>


## How to use:

`$ backlight-adjust [15-255]` <br>
`$ backlight-adjust` <br>
<br>

Help menu: <br>
`$ backlight-adjust -help` <br>
<br>
